# Proyecto Concurso de Radio

Este programa en C simula un concurso de radio donde los oyentes votan por sus canciones favoritas.

## Descripción

El programa registra votos, calcula la popularidad de las canciones, asigna puntos a los oyentes y determina un ganador.

## Instrucciones de Uso

1.  Clona este repositorio.
2.  Compila el código: `gcc concurso.c -o concurso`
3.  Ejecuta el programa: `./concurso`
4.  Ingresa los votos de los oyentes cuando se te solicite.
